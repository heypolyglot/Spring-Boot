# Spring Boot & React(AWS) + Python AI 기반 프로젝트 학습

## 개요

**풀스택 웹 개발 + DevOps + 클라우드 + AI**

현대 소프트웨어 개발 트렌드에 부합하는 종합적인 개발 역량 확보

### 핵심 학습 목표
- **풀스택 개발**: Java/Spring Boot + React 웹 애플리케이션 구축
- **클라우드 & 인프라**: AWS, Terraform, Kubernetes 기반 운영
- **AI 서비스**: Python LangChain, RAG, MCP 서버 개발
- **DevOps**: CI/CD 자동화 및 무중단 배포

## 커리큘럼

### 1단계: 객체지향 JAVA 프로그래밍 (224시간)
**학습 목표**: Java 언어와 OOP 원리 마스터  
**주요 내용**: 클래스, 상속, 추상화, 인터페이스, 컬렉션, 람다, 예외 처리 등 자바 핵심 기능

### 2단계: RDB활용 (80시간)
**학습 목표**: SQL 및 관계형 DB 활용  
**주요 내용**: SQL CRUD, 조인, 인덱스, 트랜잭션, MySQL 사용

### 3단계: 데이터아키텍쳐 (16시간)
**학습 목표**: DB 설계 및 데이터 흐름 구조화  
**주요 내용**: ERD 설계, 정규화, 관계 모델링, 데이터 흐름 설계 및 설계 문서화

### 4단계: 자바 기반서버 구축 (112시간)
**학습 목표**: Java 기반 백엔드 아키텍처 구성 실습  
**주요 내용**: Servlet, JSP, JDBC 기반 서버 구성, MVC 구조 이해, HTTP 통신 구조, WAS 구성

### 5단계: AWS와 Docker 이해 (40시간)
**학습 목표**: 클라우드 인프라의 기초 및 컨테이너 기술 이해  
**주요 내용**: AWS 핵심 서비스 (EC2, S3, IAM, RDS 등), Docker 이미지 생성, 컨테이너 실행 및 관리, 개발→배포 흐름에서의 인프라 구성 방식 파악

### 6단계: AWS기반 Spring REST서버구축 (120시간)
**학습 목표**: Spring 기반 RESTful API 서버를 AWS에 구축  
**주요 내용**: Spring Boot, Spring MVC 구조, REST API 설계, Controller-Service 구조, EC2 기반 서버 배포 실습, 환경 구성

### 7단계: 웹표준 UI 구현 (40시간)
**학습 목표**: 시맨틱 마크업과 접근성 고려한 HTML/CSS UI 구현  
**주요 내용**: HTML5, CSS3, 시맨틱 태그, 반응형 디자인, 웹 접근성 기본

### 8단계: 바닐라 JS (88시간)
**학습 목표**: JS 문법 및 DOM 조작 이해  
**주요 내용**: ES6+, 이벤트 처리, 비동기 처리(fetch, promise), DOM 조작, 컴포넌트화 기초

### 9단계: React/Node UI서버 구현 (80시간)
**학습 목표**: SPA UI 구현 및 간단한 Node 서버 구축  
**주요 내용**: React 컴포넌트, 라우팅, 상태 관리, Express 기반 Node API 서버 구현

### 10단계: Python 기반 AI/LLM 개발 (100시간)
**학습 목표**: LangChain, LangGraph를 활용한 AI 시스템 개발 및 RAG 구현  
**주요 내용**: 
- Python 기본 문법 및 AI 라이브러리 활용
- LangChain을 활용한 LLM 애플리케이션 개발
- LangGraph를 활용한 복잡한 AI 워크플로우 구축
- RAG (Retrieval-Augmented Generation) 시스템 설계 및 구현
- 벡터 DB 활용 (ChromaDB, Pinecone 등)
- MCP (Model Context Protocol) 서버 개발
- AI 마이크로서비스 아키텍처 구성

### 11단계: IaC와 Terraform 기초 (60시간)
**학습 목표**: 코드로 인프라를 관리하는 Infrastructure as Code 이해  
**주요 내용**: 
- Terraform 문법 및 구조 (provider, resource, variable)
- AWS 리소스 자동화 (VPC, EC2, RDS, S3)
- 인프라 버전 관리 및 상태 관리
- 환경별 인프라 구성 (dev, staging, prod)

### 12단계: Kubernetes 설계 및 구축 (96시간)
**학습 목표**: 컨테이너 오케스트레이션을 통한 확장 가능한 애플리케이션 배포  
**주요 내용**: 
- Kubernetes 기본 개념 (Pod, Service, Deployment, ConfigMap)
- EKS(Elastic Kubernetes Service) 클러스터 구축
- 마이크로서비스 아키텍처 설계 및 배포
- Ingress, LoadBalancer를 통한 트래픽 관리
- Horizontal Pod Autoscaler, Cluster Autoscaler

### 13단계: DevOps 파이프라인 구축 (72시간)
**학습 목표**: CI/CD 파이프라인을 통한 자동화된 배포 시스템 구축  
**주요 내용**: 
- GitHub Actions를 활용한 CI/CD 파이프라인
- Docker 이미지 자동 빌드 및 레지스트리 푸시
- Terraform을 통한 인프라 자동 프로비저닝
- Kubernetes 배포 자동화 (GitOps 패턴)
- 모니터링 및 로깅 (Prometheus, Grafana, ELK Stack)

### 14단계: AWS 기반 실무프로젝트 (MSA + AI 통합) (336시간)
**학습 목표**: 모든 기술을 종합한 실전형 AI 기반 서비스 구축  
**주요 내용**: 
- 요구 분석, ERD, API 명세 (AI 서비스 포함)
- Spring + React + Python AI + RDB + Docker 조합
- 마이크로서비스 아키텍처 (Java 백엔드 + Python AI 서비스)
- Terraform으로 AWS 인프라 구성 (VPC, EKS, RDS, ALB, GPU 인스턴스)
- Kubernetes 클러스터에 AI 마이크로서비스 배포
- LangChain 기반 RAG 시스템을 활용한 지능형 서비스
- MCP 서버를 통한 AI 모델 통신 및 컨텍스트 관리
- CI/CD 파이프라인을 통한 AI 모델 및 서비스 무중단 배포

#### 기술별 핵심 포인트

**Java/Spring 학습 시**
- 처음엔 `@RestController`, `@Service`, `@Repository`만 이해
- JPA는 `@Entity`, `@Id`, `findAll()`, `save()` 정도면 시작 가능
- 복잡한 관계 매핑은 나중에 필요할 때 학습

**React 학습 시**
- useState, useEffect, axios만 먼저 익히기
- 컴포넌트 재사용은 두 번째 프로젝트부터 고민
- CSS 프레임워크(Material-UI, Tailwind) 적극 활용

**Python AI 학습 시**
- FastAPI + LangChain 조합으로 시작
- 로컬 LLM(Ollama)으로 먼저 테스트, 안정화되면 OpenAI API
- 벡터 DB는 ChromaDB로 시작 (설치 간단)

**AWS/Kubernetes 학습 시**
- AWS는 EC2, RDS, S3만 먼저 익히기
- Kubernetes는 로컬 minikube → AWS EKS 순서
- Terraform은 간단한 EC2 생성부터 시작
